# SIm
This is a patcher for [Simplinfo by justaguy1091-bit](https://github.com/justaguy1091-bit/simplinfo)

It is called Simpl Info Improved purely because of the "VIM/VI Improved" naming scheme. I would not call this project an improvement under any circumstances.

## Methodology

This project is a patcher for Simplinfo, and is not a standalone version. The goal is to take the latest version of the Simplinfo project and add some features to it.

Notably, adding the distro icons from Fast Fetch...
